1.StackOverFlow

   Stack OverFlow merupakan situs tanya-jawab yang favorit tentang programming , bagi programming pemula maupun yang sudah profesional. Situs ini telah banyak membantu programmer yang sedang menghadapi masalah codingan dan berharap menemukan titik terang dari permasalahan yang sedang di alaminya.

2.Git

   Git adalah merupakan tool atau Version Control System (VCS) berbasis open source yang berguna dalam proyek pengembangan perangkat lunak. Git ini banyak dipilih karena memliki kinerja, keamanan, dan fleksibilitas yang maksimal sehingga memudahkan kawan-kawan untuk mengatur versi dari source code yang sedang dibangun.

3.Google

   Google merupakan mesin pecarian web yang harus digunakan untuk mencari solusi maupun jawaban apapun itu. bagi kebanyakan programmer mau itu pemula ataupun yang sudah profesional, merupakan cara pertama yang digunakan untuk mencari serta menemukan solusi akan permasalahan codingan yang di hadapi.

4.GitHub

   GitHub adalah platform layanan berbasis Version Control System (VCS), yang paling populer digunakan develover untuk menyimpan berbagai file source code atau repository secara gratis dan juga menyediakan software.
