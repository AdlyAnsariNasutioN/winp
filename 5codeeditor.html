1.ATOM

   Atom adalah text editor yang bersifat free source dan open source bagi MacOS, Linux, dan Microsoft Windows. Atom di lengkapi dengan plug-in yang di tulis dalam Node.JS yang dikembangkan oleh GitHub, dan Atom merupakan aplikasi desktop yang dibuat memakai teknologi web.

2.SUBLIME TEXT

   Sublime Text adalah sebuah teks editor yang sangat canggih untuk coding, mark-up dan lain sebagainya.

3.VISUAL STUDIO CODE

   Visual Studio Code adalah code editor source code yang dikembangkan oleh Microsoft untuk Windows, Linux, dan MacOS. Ini termasuk dukungan untuk debugging, GIT Control yang disematkan, penyororotan sintaks, penyelesaian kode cerdas, cuplikan, dan kode refactoring.

4.NOTEPAD++

   NotePad++ adalah suatu text editor yang berjalan pada operating System(OS) Windows. NotePad++ disini menggunakan komponen-komponen Scintilla agar dapat menampilkan dan menyunting text dan berkas source code berbagai bahasa pemrograman.

5.BRACKETS

   Brackets adalah code editor yang secara khusus dikembangkan untuk tujuan Web Design dan Front-end Development.
