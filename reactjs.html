   React Js adalah sebuah library yang digunakan untuk membangun user interface atau tampilan website. React Js menggunakan HTML, CSS dan JavaScript dalam membangun sebuah user interface web. Ada banyak website yang telah menggunakan React JS dalam membangun User interface. Salah satunya web popular yang menggunakan React JS adalah Tokopedia dan Juga Traveloka.

Berikut adalah kelebihan dari React JS :

  • React JS mudah dipelajari dan dikuasai develover
    • perlu mempelajari dasar-dasar teknologi tersebut dan memahami fungsinya agar dapat memaksimalkan fitur-fitur yang tersedia.
  • Mendukung reusability komponen
    • React JS memungkinkan kawan-kawan untuk menggunakan kembali komponen yang telah dikembangkan ke aplikasi lain yang menggunakan fungsi yang sama.
  • Menghadirkan tools
    • develover langsung dari sumbernya ini memungkinkan kawan-kawan untuk mengamati hierarki komponen reaktif dan memeriksa status serta alat bantu komponen yang ada.

    Dan berikut adalah kekurangan dari React JS :

  • Pada view layer
    • Untuk membangun suatu aplikasi yang besar harus mengatur sendiri kebutuhan aplikasi lainnya seperti router, struktur aplikasi, data layer, dan event system (kecuali even DOM).
  • Dokumentasi kurang bagus
    • Dokumentasi React JS kurang bagus, karena informasi ang diberikan tidak lengkap.
  • Dukungan Browser
    • Browser pada React JS hanya boleh menggunakan Browser versi baru yaitu internet Explorer versi 9 ke atas.
