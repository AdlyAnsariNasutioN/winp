Mengenal lebih Jauh Tentang Back End?

   Back End adalah orang yang bekerja dibalik layar, semua yang tidak di lihat oleh pengguna akhir atau berinteraksi langsung. Back End focus pada database, Scripting, dan arsitektur dari sebuah website.

   Pada umum-nya, orang- orang yang bekerja sebagai BACK END DEVELOVER dikenal sebagai pemecah masalah terbaik kawan-kawan, karena mereka di nilai selalu memakai pikiran logis dalam menjalankan pekerjaaan.

   Secara alur, kerja Back End dimulai dari tampilan layar muka WEB yang dikembangkan oleh Front End kawan-kawan. Dari layout tersebut, mereka akan memilih system yang cocok atau sesuai apa yang di inginkan oleh client tersebut.

   Maka dari itu, tidak mengherankan apabila kerja keduanya tidak bisa dipisahkan kawan-kawan , Udah Klop lah gitu hehheheh... Untuk lebih detailnya, Berikut ini adalah tugas Seorang BACK END DEVELOVER Simak ya kawan-kawan sekalian:

  • Menciptakan, Mengintegrasikan, dan Mengola Database.
  • Bekerja dengan FrameWork Back End untuk membangun software Server –Side
  • Teknologi server Web Integritas Cloud Computing.
  • Mengelola dan mengembangkan content management system (CMS)
  • Integritas API
  • Pengaturan keamanan dan pencegahan peretasan (hacker)
  • Melaporkan analisis dan statistic

    •    Sebelum itu kawan-kawan, yang harus kawan-kawan harus tahu adalah skil dan teknis jika ingin menjadi seorang Back End Develover yaitu :

  • Pahami apa itu server
  • Pelajari database & HTTP
  • Dan kawan-kawan harus tahu apa itu API
    • Untuk bahasa Pemrograman yang kawan-kawan perlu kuasai adalah yaitu :

    Python
