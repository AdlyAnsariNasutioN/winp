   Banyak profesi khususnya di bidang Teknologi yang di dalamnya menggunakan source code / codingan untuk membuat suatu aplikasi ataupun website itu adalah hal lumrah yang kita temukan di dalam suatu perkembangan Teknologi saat ini. Nah Dari pada kawan-kawan menunggu terlalu lama, langsung saja simak penjelasan dibawah ini ya kawan-kawan.

-) 1. UI/UX DESIGNER

   UI/UX Designer secara garis besar bertugas untuk membuat desain sebuah produk seperti website atau aplikasi yang nyaman dan user frindly.

-) 2. SEO/SEM SPECIALIST

   SEO dan SEM specialist bertugas untuk meningkatkan rangking sebuah website pada mesin pencarian. Bedanya, SEO tidak menggunakan iklan berbayar dan SEM itu berbayar dan tetap melibatkan SEO untuk hasil akhir.

-) 3. PRODUCT MANAGER

   Product Manager fokus pada strategis bisnis, desain, dan bertanggung jawab pada product launching tiba. walaupun tidak mengharuskan kemampuan coding, tapi punya basic coding akan sangat membantu pekerjaan kawan-kawan.

-) 4. PROJECT MANAGER

    Peran Project manager dibutuhkan untuk memastikan proyek lancar saat membuat produk. mereka memimpin jalannya proyek dan membina hubungan baik dengan berbagai pihak seperti Product Manager dan Engineer.

-) 5. TECHNICAL WRITER

    Technical Writer bertugas untuk menyusun komunikasi teknis. peran mereka dibutuhkan dalam pembuatan buku panduan dibidang software komputer, robot, keuangan, medis, bioteknologi dan sebagainya.

-) 6. UX WRITER

    Peran UX Writer sangat penting dalam sebuah product. mereka menulis setiap kata-kata yang biasanya kamu lihat pada website atau aplikasi. Menulisnya pun perlu kemampuan khusus supaya pengalaman user menjadi menyenangkan kawan-kawan.
