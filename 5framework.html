1.Semantic UI

   Framework CSS yang fokus kepada penggunaan human-friendly HTML. Human-friendly HTML memudahkan develover pemula membangun desain front-end kekinian.

2.Foundation

   Framework dengan sistem open-source berbasis SASS. Framework ini menawarkan berbagai elemen yang modern. fleksibel dan mudah untuk dikustominasi.

3.Bulma

   Bulma hanya menggunakan komponen berbasis CSS (tanpa JavaScript). Daya tariknya ada pada penggunaanya yang mudah, responsif, bersifat open-source, dan bekerja secara optimal di segala perangkat.

4.Bootstrap

   Bootstrap telah mengupgrade versi ke lima (5.0). mereka dengan segudang tambahan fitur, mulai dari skema warna baru, kelas, sistem grid,, layout modern dan sebagainya. untuk inspirasi, Bootstrap juga menampilkan jejeran contoh desain dari aneka aplikasi dan website.

5.Ulkit

   Ulkit populer dengan deretan desainnya yang simpel, minimalis dan elegan. kawan-kawan bisa membuat interface dengan set icon SVG, komponen yang lengkap. style yang modern, dan mudah dikustomisasi.
